## 📞 技术支持

### 访问方式
- **Web 界面**: http://127.0.0.1:5000
- **API 文档**: http://127.0.0.1:5000/api/targets
- **配置文件**: cymind.yaml

### 故障排除
1. **端口占用**: 修改配置文件中的端口
2. **依赖缺失**: `pip3 install -r requirements.txt`
3. **工具未安装**: 安装 nmap 等扫描工具
4. **配置错误**: 参考 cymind.yaml.example

### 日志查看
```bash
# 应用日志
tail -f logs/cymind.log

# 错误日志
tail -f logs/error.log

# 扫描日志
tail -f logs/scan.log
```
